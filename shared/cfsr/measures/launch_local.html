🚀 CFSR Performance Dashboard

⚠️ Shiny Server Required
To view the interactive dashboard, you need to run a local Shiny server. Follow the steps below.

Step 1: Open R or RStudio

Launch R or RStudio on your computer.

Step 2: Run the Shiny App

Copy and paste this command into the R console:

shiny::runApp("D:/repo_childmetrix/cm-reports/shared/cfsr/measures/app_national", port = 3838)

You should see a message like:

Listening on http://127.0.0.1:3838

Step 3: Access the Dashboard

Choose one of these options:

Option A: Direct Access
Click this link (after running the app in R): Open Dashboard

Option B: Via ChildMetrix Platform
1. Keep the R session running
2. Open the platform: ChildMetrix Platform
3. Click "Performance" in the secondary navigation

Alternative: Update for File Protocol

If you want the dashboard to work without running a server, you would need to:

  • Convert the Shiny app to static HTML (loses interactivity)
  • Use a different framework (not Shiny)
  • Set up a proper web server (Shiny Server, Apache, nginx)
